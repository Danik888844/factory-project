# factory-project

## Project setup
```
yarn install or npm install
```

### Compiles and hot-reloads for development
```
yarn serve or npm run serve
```

### Compiles and minifies for production
```
yarn build
```

### Don't forget to start json-server
```
npx json-server --watch data/db.json
```
